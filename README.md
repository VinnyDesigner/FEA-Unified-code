# Full-Stack Application

This is a full-stack application built with React (Vite) and Node.js (Express).

## Tech Stack
- **Frontend**: React.js, Vite, React Router, Axios, Tailwind CSS, Zustand
- **Backend**: Node.js, Express.js, MongoDB (Mongoose)

## Setup Instructions

1. Clone the repository.
2. Run `npm run install:all` in the root directory to install all dependencies for both frontend and backend.
3. Copy `.env.example` to `.env` in the `server` directory (and adjust if needed).
4. Start the application:
   ```bash
   npm start
   ```

This will run both the frontend and backend concurrently.
